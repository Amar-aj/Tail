## ?? Quick Start Commands

```bash
# Create new project
dotnet new blazorserver -o MyApp
cd MyApp

# Install Tail.Blazor
dotnet add package Tail.Blazor.Core
dotnet add package Tail.Blazor.Buttons
dotnet add package Tail.Blazor.Forms

# Run
dotnet run
```

Visit `/test` to see components in action!
