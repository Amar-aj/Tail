## Running the Documentation

```bash
cd docs/Tail.Blazor.Docs
dotnet run
```

The site will be available at `https://localhost:5001` or `http://localhost:5000`.

## Pages

- `/` - Home page
- `/getting-started` - Installation and setup guide
- `/components` - Component overview
- `/components/buttons` - Button component documentation
- `/components/forms` - Form components documentation
- `/theming` - Theme customization guide
- `/examples` - Live examples
- `/api` - API reference

## Structure

- `Pages/` - Documentation pages

- `wwwroot/` - Static files
