# Tail.Blazor.RangeValidator

Independent NuGet package for the TailRangeValidator component.

## Installation

```bash
dotnet add package Tail.Blazor.RangeValidator
```

## Usage

```razor
@using Tail.Blazor.RangeValidator

<TailRangeValidator Value="@ageValue" Min="18" Max="100" />
```

## Features

- Range validation
- Min/max constraints
- Custom error message
- Full theme support with CSS variables
- MAUI Blazor Hybrid compatible

## Package Size

~1 KB (ultra-lightweight)

## Dependencies

- Tail.Blazor.Core (required)
