# Tail.Blazor.Scheduler

Independent NuGet package for the TailScheduler component.

## Installation

```bash
dotnet add package Tail.Blazor.Scheduler
```

## Usage

```razor
@using Tail.Blazor.Scheduler

<TailScheduler CurrentDate="@DateTime.Now" Events="@events" />
```

## Features

- Calendar scheduler
- Event display
- Month view
- Full theme support with CSS variables
- MAUI Blazor Hybrid compatible

## Package Size

~6 KB (ultra-lightweight)

## Dependencies

- Tail.Blazor.Core (required)
