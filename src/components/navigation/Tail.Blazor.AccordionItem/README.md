# Tail.Blazor.AccordionItem

Independent NuGet package for the TailAccordionItem component.

## Installation

```bash
dotnet add package Tail.Blazor.AccordionItem
```

## Usage

```razor
@using Tail.Blazor.AccordionItem

<TailAccordionItem Title="Section 1" IsExpanded="@isExpanded">
    Content here
</TailAccordionItem>
```

## Features

- Individual accordion item
- Expand/collapse animation
- Full theme support with CSS variables
- MAUI Blazor Hybrid compatible

## Package Size

~2 KB (ultra-lightweight)

## Dependencies

- Tail.Blazor.Core (required)
