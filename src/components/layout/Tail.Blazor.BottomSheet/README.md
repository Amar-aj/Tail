# Tail.Blazor.BottomSheet

Independent NuGet package for the TailBottomSheet component.

## Installation

```bash
dotnet add package Tail.Blazor.BottomSheet
```

## Usage

```razor
@using Tail.Blazor.BottomSheet

<TailBottomSheet IsVisible="@showSheet" Size="BottomSheetSize.Md">
    Sheet content
</TailBottomSheet>
```

## Features

- Bottom sheet modal
- 4 sizes (Sm, Md, Lg, Full)
- Backdrop click to close
- Full theme support with CSS variables
- MAUI Blazor Hybrid compatible

## Package Size

~3 KB (ultra-lightweight)

## Dependencies

- Tail.Blazor.Core (required)
