# Tail.Blazor.CollapsibleMenu

Independent NuGet package for the TailCollapsibleMenu component.

## Installation

```bash
dotnet add package Tail.Blazor.CollapsibleMenu
```

## Usage

```razor
@using Tail.Blazor.CollapsibleMenu

<TailCollapsibleMenu Title="Menu" IsExpanded="@isExpanded">
    Menu items
</TailCollapsibleMenu>
```

## Features

- Collapsible menu
- Expand/collapse animation
- Full theme support with CSS variables
- MAUI Blazor Hybrid compatible

## Package Size

~2 KB (ultra-lightweight)

## Dependencies

- Tail.Blazor.Core (required)
