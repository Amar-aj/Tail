## üìù Template for Creating New Component Packages

### Step 1: Create Folder Structure
```powershell
New-Item -ItemType Directory -Path "src\components\{category}\Tail.Blazor.{ComponentName}" -Force
```

### Step 2: Create .csproj File
```xml
<Project Sdk="Microsoft.NET.Sdk.Razor">
  <PropertyGroup>
    <TargetFramework>net8.0</TargetFramework>
    <Nullable>enable</Nullable>
    <ImplicitUsings>enable</ImplicitUsings>
    <Version>1.0.0</Version>
    <Authors>Tail.Blazor Core Team</Authors>
    <Company>Tail.Blazor</Company>
    <Product>Tail.Blazor {ComponentName}</Product>
    <Description>Independent {component} component package for Tail.Blazor - Ultra-lightweight ({size} KB). Install only what you need.</Description>
    <PackageLicenseExpression>MIT</PackageLicenseExpression>
    <PackageProjectUrl>https://tailblazor.com</PackageProjectUrl>
    <RepositoryUrl>https://github.com/tailblazor/tailblazor</RepositoryUrl>
    <PackageTags>blazor;tailwind;ui;component;{tags};lightweight;performance;independent</PackageTags>
    <GenerateDocumentationFile>true</GenerateDocumentationFile>
    <LangVersion>latest</LangVersion>
    <EnableTrimAnalyzer>true</EnableTrimAnalyzer>
    <PublishTrimmed>true</PublishTrimmed>
  </PropertyGroup>
  <ItemGroup>
    <SupportedPlatform Include="browser" />
  </ItemGroup>
  <ItemGroup>
    <PackageReference Include="Microsoft.AspNetCore.Components.Web" Version="8.0.0" />
  </ItemGroup>
  <ItemGroup>
    <ProjectReference Include="..\..\core\Tail.Blazor.Core\Tail.Blazor.Core.csproj" />
  </ItemGroup>
</Project>
```

### Step 3: Create _Imports.razor
```razor
@using Microsoft.AspNetCore.Components.Web
@using Tail.Blazor.Core
```

### Step 4: Create Component File
- Create `Tail{ComponentName}.razor` file
- Inherit from `TailComponentBase`
- Use CSS variables for theming
- Follow MAUI compatibility guidelines

### Step 5: Create Enums/Models (if needed)
- Create component-specific enums
- Create component-specific models
- Keep them independent (no shared dependencies)

### Step 6: Create README.md
- Installation instructions
- Usage examples
- Features list
