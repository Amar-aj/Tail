# Tail.Blazor.Sparkline

Independent NuGet package for the TailSparkline component.

## Installation

```bash
dotnet add package Tail.Blazor.Sparkline
```

## Usage

```razor
@using Tail.Blazor.Sparkline

<TailSparkline Data="@sparklineData" Width="100" Height="30" />
```

## Features

- Mini sparkline chart
- Inline data visualization
- SVG-based rendering
- Full theme support with CSS variables
- MAUI Blazor Hybrid compatible

## Package Size

~2 KB (ultra-lightweight)

## Dependencies

- Tail.Blazor.Core (required)
