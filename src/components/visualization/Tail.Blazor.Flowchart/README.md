# Tail.Blazor.Flowchart

Independent NuGet package for the TailFlowchart component.

## Installation

```bash
dotnet add package Tail.Blazor.Flowchart
```

## Usage

```razor
@using Tail.Blazor.Flowchart

<TailFlowchart Nodes="@flowchartNodes" Connections="@connections" Width="800" Height="600" />
```

## Features

- Flowchart visualization
- Nodes and connections
- SVG-based rendering
- Full theme support with CSS variables
- MAUI Blazor Hybrid compatible

## Package Size

~5 KB (ultra-lightweight)

## Dependencies

- Tail.Blazor.Core (required)
